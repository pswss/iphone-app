스케줄 앱 · 디자인 목업

Liquid Glass · 시안 v2

실제 앱이 아니라 디자인 미리보기(웹)입니다. 무지개 바의 전진하는 선과 현재 칸 강조는 하루를 빠르게 돌려 보여줍니다. 색은 일정 시간 순서대로 자동(빨→보)이고, 포인트 컬러는 라이트=화이트 / 다크=남색으로 고정했습니다.

1 · 메인 스케줄 & 일정 추가

2 · 잠금화면 Live Activity (당일 일정 바)

이 바가 어떻게 동작하나

  • 빨주노초파남보 세그먼트 — 일정 시간 순서대로 색이 자동 지정 (유저가 고르지 않음)
  • 전진하는 흰 선 — 현재 시각. iOS에선 ProgressView(timerInterval:)로 코드 없이 시스템이 스스로 전진
  • 현재 칸 강조 — 진행 중인 칸만 떠오르고, 일정 이름은 바 아래에 "현재 일정 · …"로 표시
  • 위 미리보기는 하루를 빠르게 돌려 선이 지나가며 강조가 옮겨가는 걸 보여줍니다

참고: 앱을 오래 닫아두면 "강조 칸이 다음으로 점프"하는 마지막 연출만 백그라운드 제약이 있어, 앱 진입/알림 탭 시 즉시 보정됩니다. (완벽 자동전환은 추후 푸시로)

3 · 다이나믹 아일랜드 3상태

4 · AI로 일정 만들기 (Claude 연동)

AI 일정 생성은 이렇게 동작해요

  • 입력한 자연어를 Claude API(Anthropic)로 보냄
  • Claude가 구조화된 일정 데이터(제목·시작·종료)로 변환해 반환
  • 앱이 결과를 미리 보여주고, 확인하면 SwiftData에 저장 + iCloud 동기화
  • 색상은 평소처럼 시간 순서대로 자동 지정

키 보관: API 키는 코드에 넣지 않고 기기 Keychain에 저장. (정식 배포 시엔 키를 직접 노출하지 않는 중계 방식이 더 안전 — 추후 단계)
